### 2. 细胞呼吸三个阶段
* 第一阶段：糖类、脂肪酸和氨基酸氧化生成乙酰辅酶A
* 第二阶段：乙酰辅酶A通过柠檬酸循环彻底氧化
* 第三阶段：NADH和FADH2中的电子通过电子传递链传递给氧气，产生水和质子梯度，用于ATP生成
